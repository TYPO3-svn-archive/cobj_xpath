Installation
------------

Requirements
^^^^^^^^^^^^

- You need at least PHP 5.2+

- You need libxml and SimpleXML enabled

- The extension has been tested on TYPO3 versions 4.5 up to 6.1

Installation
^^^^^^^^^^^^

Simply import it from TER and install. Nothing else needs to be done.
